====== Data Operation - List ======


===== About =====

''list'' is an [[:docs:flow:intermediate|intermediate operation]] that gathers [[docs:resource:attribute|attributes]] of the [[:docs:flow:input|input resources]] and returns them in a [[:docs:resource:tabular|tabular fashion]]


===== Cli =====

It's called the ''list'' operation because it's the operation used in the [[:docs:tabul:data:list|list command]].


===== Arguments =====

^ Name ^ Default ^ Description ^
| attributes | [[:docs:resource:name|name]], [[:docs:resource:media-type|media type]] | A list of attributes name to return |
