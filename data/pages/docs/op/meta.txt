====== Data Operation - List ======


===== About =====
''Meta'' are [[op|data operation]] that shows [[:docs:resource:metadata|metadata]].





===== Tabular Format Type =====

Data resources displaying metadata may have the following [[docs:resource:tabular|tabular format]] type:
  * ''Record'': a table where
      * each column represents the value of an attribute
  * ''Form'': a table with the following set of fixed columns:
      * attribute name
      * attribute description
      * value
      * and default value

===== Avaibility =====

==== Intermediate Operations ====

The following [[:docs:flow:intermediate|intermediate operations]] add [[docs:resource:attribute|data resources attributes]] from their  [[:docs:flow:input|inputs]] in a `Record` fashion
  * [[list]]
  * [[enrich|Enrich]] with the [[:docs:generator:meta|meta data supplier]]
