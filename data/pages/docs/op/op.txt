====== Data Operations ======



{{ :docs:op:undraw_data_processing.svg?600&ratio=16:9 |}}

===== About =====
The ''data operations'' are all functions, operations, actions that you can perform on a [[docs:resource:resource|data resources]]. 

They can be used in a [[:docs:flow:step|step]] of a [[:docs:flow:pipeline|pipeline]].

This section lists all ''data operations'' supported.

<note>
Because all data structures are represented logically as a [[docs:resource:resource|data resource]] (file, table, ...), they all share the same common data vocabulary.
</note>
