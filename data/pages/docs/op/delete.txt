====== Data Operation - Delete ======


===== About =====
''delete'' is an alias of the [[transfer|transfer operation]] where:
  * the operation is ''delete''.
  * the [[:docs:flow:input|input]] defines the records to delete in the [[:docs:flow:target|target]]





===== Cli =====


The ''delete'' operation is also available with the [[docs:tabul:data:delete|data delete command]].
