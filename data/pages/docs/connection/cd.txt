====== System Connection - Cd (Current Directory) ======


===== About =====

The ''cd'' [[connection|connection]] is a [[built-in|built-in connection]] that points to the current [[docs:resource:directory|directory]] of the [[docs:system:local|local file system]].

This is the [[connection#default|default connection]] if this is not a [[docs:app:app|app]]

===== Info =====
To get the path of the current directory, with the [[docs:tabul:connection:info|connection info]] command
<unit>
<code bash>
tabul connection info cd
</code>
