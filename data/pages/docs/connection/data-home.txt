====== Data Home (XDG_DATA_HOME) Connection ======


===== About =====

''data home'' is [[built-in|built-in connection]] that points to the [[https://specifications.freedesktop.org/basedir-spec/latest/|xdg-data-home]] [[docs:resource:directory|directory]] of the [[docs:system:local|local file system]].

===== Usage =====

It's where application data is stored by default.

For instance, in a pipeline, data resources with [[:docs:flow:error_handling#parking|errors are parked]] in this location.




===== Modification =====
On Linux, you can change this directory by setting the ''XDG_DATA_HOME'' [[:docs:conf:os|env]]

With the [[docs:tabul:connection:info|connection info]] command
