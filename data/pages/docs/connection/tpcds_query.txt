====== Tpcds_query Connection ======


===== About =====
''tpcds_query'' is  a [[docs:system:local|local file system]] [[connection|connection]] that points to the directory where the [[docs:resource:sql_select|queries]] of the [[docs:system:tpcds|TPC-DS]] benchmark.

It's one of the [[howtos|howto's connections]] created at the installation of ''Tabulify''.


===== Location =====

This directory is a subdirectory of the [[docs:tabul:home|installation directory]].

With the [[docs:tabul:connection:info|connection info]] command:
<unit>
<code bash>
tabul connection info tpcds_query
</code>
<console>

</code>
