====== Tmp (Temporary Runtime directory) Connection ======


===== About =====

''tmp'' is a [[built-in|built-in connection]] that points to the runtime temporary [[docs:resource:directory|directory]] of the [[docs:system:local|local file system]].


===== Value =====


<code>
RUNTIME_DIR/tabul
</code>
where:
  * ''RUNTIME_DIR'' is the value of the [[https://specifications.freedesktop.org/basedir-spec/latest/|XDG_RUNTIME_DIR]] [[docs:conf:os|os env]] if found otherwise it's the operating system temporary directory.


Example of Runtime Temporary Directories:
  * Linux: ''/tmp''
  * Windows WSL Linux: ''/mnt/wslg/runtime-dir''
  * Windows: ''%TEMP%''
===== Usage =====

This is the default connection for the logs of [[:docs:op:execute|execution]] and other temporary runtime data.
