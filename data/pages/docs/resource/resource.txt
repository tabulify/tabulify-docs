====== Tabulify - Data Resources ======

{{ :docs:resource:undraw_environment.svg?600&ratio=16:9 |}}

===== About =====
A [[:docs:system:file|file system]] manipulate [[:docs:resource:file|files]], a [[docs:system:database:database|database]] manipulates ''sql objects'' ([[docs:resource:sql_table|table]], [[docs:resource:sql_view|view]], ...).

Tabulify regroups and manipulates all of them all under the term ''data resource''.


This section lists all data resources supported.

===== Type =====

A data resource can be of the following type:

^ Type ^ Name ^ Example ^ Description ^
| ''static'' | ''static resource'' | [[sql_table|table]], [[file|file]], ... | The content is stored on disk at rest |
| ''runtime'' | [[runtime|runtime resource]] | [[request|request]], [[docs:resource:script|script]] | The content is generated at runtime |
| ''container'' | [[container|container resource]] | [[sql_schema|sql schema]], [[directory|directory]] | Resources that contain data resources |

===== Definition =====

A ''resource'' is uniquely defined by:
  * a [[data_uri|data uri]] to locate the resource
  * and optionally:
    * a [[docs:resource:media-type|media type]] to define the type of resource
    * a [[docs:resource:data-definition|data definition]] to define the [[docs:resource:attribute|attributes]] of the resources


===== List by System =====

==== Database ====

List of resources for [[docs:system:database:database|database system]]
  * [[sql_table|Sql Table]]
  * [[sql_view|Sql View]]
  * [[sql_schema|Sql Schema]]

    * [[:docs:resource:sql_dml|DML (SELECT/UPDATE/INSERT/MERGE)]]
    * [[:docs:resource:sql_ddl|DDL (CREATE/ALTER/DROP)]]
    * [[:docs:system:database:sql_procedure|SQL Procedures (CALL)]]
