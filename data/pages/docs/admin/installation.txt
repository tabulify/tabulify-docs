==== Docker (Linux, Windows, macOS) ====

With [[https://github.com/tabulify/tabulify/pkgs/container/tabulify|docker releases]]

```bash
docker run \
  --rm \
  -v $(pwd):/workspace \
  ghcr.io/tabulify/tabulify-alpine:latest \
  tabul --version
```
