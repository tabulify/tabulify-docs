====== Sqlite - Boolean ======


===== Boolean =====
[[sqlite|Sqlite]] does not support natively the ''BOOLEAN'' data type. 

It recommends to save it as an integer ''0'' (false) and ''1'' (true). 
See [[https://sqlite.org/datatype3.html#boolean_datatype|Sqlite boolean documentation]]
