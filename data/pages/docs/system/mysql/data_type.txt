===== Note =====

==== Text Data Type ====
See [[docs:system:mysql:text|]]

==== Integer Data Type ====

We support all [[https://dev.mysql.com/doc/refman/8.4/en/integer-types.html|integer types]] included the extra non standard [[https://dev.mysql.com/doc/refman/8.4/en/integer-types.html|mediumint type]].

==== Floating Point Number ====



We follow MySQL that treats ''REAL'' as a synonym for ''DOUBLE PRECISION'' (a nonstandard variation on the SQL standard).
If you use the [[https://dev.mysql.com/doc/refman/8.4/en/sql-mode.html#sqlmode_real_as_float|REAL_AS_FLOAT]] SQL mode, the tabulify behavior is unknown. It will still work with raw [[:docs:resource:sql_file|sql script]].

==== No Time Zone ====


Time zone data type does exist as MySQL [[https://dev.mysql.com/doc/refman/8.4/en/datetime.html|converts TIMESTAMP values from the current time zone to UTC for storage, and back from UTC to the current time zone for retrieval]].

In case of cross transfer, the time zone datatype are mapped to their counterpart without time zone to create a non-existing table:
  * ''timestamp with timezone'' to ''timestamp'' utc
  * ''time with timezone'' to ''time'' utc


==== Date and Time ====

We support all [[https://dev.mysql.com/doc/refman/8.4/en/date-and-time-types.html|date time type]]:
  * the SQL standard:
    * ''DATE'',
    * ''TIME'',
    * ''TIMESTAMP''
  * and the non-standard
    * ''DATETIME''
    * and ''YEAR''.

In a cross transfer from a source table with a SQL timestamp column, if the MySql table does not exist, the ''TIMESTAMP'' data type is chosen, not ''DATETIME''.

More on general time support can be seen on this [[docs:data_type:date_time|page]].
==== JSON ====

We support the [[https://dev.mysql.com/doc/refman/8.4/en/json.html|JSON]] type.

==== XML ====

''MySQL'' does not have any native support for ''XML'' data type, we map this type to the ''MySQL'' [[text#text|text (ie clob)]] format.

If you know the ''XML'' size, you can choose another [[text#text|Text data type]]

You can then use any [[https://dev.mysql.com/doc/refman/8.4/en/xml-functions.html|XML function]].


===== Default Type =====

If in a [[:docs:resource:manifest|resource manifest]], no type is specified and that the target resource does not exists. the default type used is ''varchar''
